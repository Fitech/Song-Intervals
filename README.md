# Workout Intensity Visualizer

A dynamic web application designed to accompany workout videos. It visualizes song intensity and displays motivational cues triggered by keyboard inputs.

## Controls

### Intensity Levels
Control the height and color of the intensity bar using number keys:
- **`1` - `3`**: Low Intensity (Blue/Green)
- **`4` - `7`**: Medium Intensity (Yellow/Orange)
- **`8` - `9`**: High Intensity (Red/Purple)
- **`0`**: Maximum Intensity (Red/Purple + Shake Effect)

### Motivational Triggers
Press these keys to display ephemeral motivational text:
- **`Space`**: "KEEP GOING!"

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the local URL (default: `http://localhost:3005`).
